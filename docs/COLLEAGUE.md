## 1. 这是什么？有什么用？

**seocli** 是一个网站 SEO 体检工具。

你可以把它理解成一个「网站检查员」——告诉它一个网址，它就会自动爬遍整个网站，然后告诉你：

- ✅ 标题、描述写得好不好
- ❌ 有没有页面打不开（404）
- 🔍 图片有没有加说明文字（alt）
- 📱 社交分享标签（Open Graph）有没有配齐
- ⚡ 哪些页面加载慢
- 🏷️ 有没有重复内容
- 等等...

最后给你一份结构化的「体检报告」，哪里有问题、严重程度、怎么改，一目了然。

**谁会用？** 运营、编辑、PM — 任何需要检查网站 SEO 健康度的人。你只需要说一句「帮我审计一下某某网站」，AI 自动搞定。

## 2. 怎么让 AI 部署和使用（复制下面这段话发给 AI）

> **这行字不用复制：** 这条消息告诉 AI 去哪里下载、怎么配置、怎么用。复制下面整段发给 AI 就行。

```
我需要你帮我部署 seocli 这个 SEO 审计工具。

## 部署步骤

1. **安装**
   运行：pip install seocli

2. **注册 MCP**
   运行：seocli setup
   这会把 seocli 注册到 ~/.claude/mcp.json，让 AI 助手可以直接调用它。

3. **验证**
   运行：seocli https://example.com --quiet --max-urls 5 --depth 1
   确认能看到 JSON 输出（有 stats、pages、issues 等字段）

## 使用方式

安装配置好之后，当我说"审计一下 XXX"或"检查 XXX 的 SEO"时：
- 直接调用 seocli_audit 或 seocli_issues_summary 工具
- 把结果整理成报告给我看，分三块：
  🔴 错误（必须修）
  🟡 警告（建议修）
  🔵 信息（知道了就行）
- 每块说清楚：什么问题、在哪个页面、严重程度
```

## 3. 日常使用

部署好之后，每次你想检查网站，**直接说人话就行**：

> "帮我审计一下我们官网的 SEO"
> "检查一下 ihcn 这个站的 SEO"
> "审计一下 https://example.com"

AI 会自动调用 seocli 去爬取和检查，然后把结果整理成你能看懂的报告给你。

如果报告里有红色错误，你可以接着问：

> "这个标题太长的问题怎么改？"
> "帮我把这些图片的 alt 文本补上"

AI 会给出具体的修改建议。
